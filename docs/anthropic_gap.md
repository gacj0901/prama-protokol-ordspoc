# Anthropic Gap Note

MITRE ATT&CK catalogs discrete techniques and is valuable for shared vocabulary, inventory and analysis. ORDSPOC does not replace ATT&CK.

ORDSPOC is inspired by the measurement gap described by Anthropic: when autonomous systems sustain orchestration over time, a catalog of discrete techniques may not capture the trajectory-level structure of the risk.

The observed gap is sustained autonomous orchestration. A path can have fewer discrete stages while carrying higher structural risk because demand and autonomy remain coupled across the trajectory.

ORDSPOC adds a measurement layer for trajectory contrast. It uses abstract, dimensionless variables and contains no attack techniques, procedures, exploit logic or operational cyber content.

This project makes no claim of affiliation with Anthropic. It uses cautious methodological framing and treats the Anthropic discussion as motivation for a structural proof-of-concept.
